# AbdCompressTools

Fast PDF management for all platforms. View, convert, compress, merge, split,
and digitally sign PDFs.

**Flutter 3.10.7+ | Dart | 50+ formats | 6 platforms**

## Features

- Viewer - View standard and password-protected PDFs
- Compress - Reduce file size
- Image to PDF - JPG, PNG, WEBP, HEIC, TIFF, GIF, BMP
- Convert - Export and create PDFs
- Edit - Add text and annotations
- Merge - Combine PDFs
- Split - Extract pages
- Sign - Digital signatures
- History - Recent files and favorites

| Yes | Yes | Yes | Yes | Yes | Yes |
