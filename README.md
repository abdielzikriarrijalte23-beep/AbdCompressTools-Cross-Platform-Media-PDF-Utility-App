## Overview

AbdCompressTools is a cross-platform media and PDF utility application built using Flutter and Dart.

The application supports PDF management, media compression, file conversion, annotations, and digital signatures across multiple platforms including Android, iOS, Windows, Linux, macOS, and Web.

---

## Features

### PDF Features
- View standard and password-protected PDFs
- Merge multiple PDFs
- Split PDF pages
- Compress PDF files
- Add text and annotations
- Digital signature support

### Media Features
- Compress images and videos
- Convert images to PDF
- Support for 50+ file formats

### Productivity Features
- File history
- Favorites
- Fast file access
- Cross-platform support
